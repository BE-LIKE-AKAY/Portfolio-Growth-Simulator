# 📈 Portfolio Growth Simulator

A **dark, futuristic portfolio growth simulator** that visualizes compound investment growth using an interactive chart.

This project is built as a **single-file web application** and runs entirely in the browser.  
No backend, no database, and no installation required — making it perfect for **GitHub Pages hosting**.

🌐 **Live Demo**  
https://be-like-akay.github.io/Portfolio-Growth-Simulator/

---

# 🚀 Features

⚡ **Daily Compounding Simulation**  
Simulate how an investment grows with daily percentage returns.

📊 **Interactive Growth Chart**  
Smooth animated visualization powered by **Chart.js**.

🌙 **Dark Futuristic UI**  
Modern dark theme interface.

📱 **Mobile Friendly**  
Fully responsive design for desktop and mobile devices.

🔗 **Shareable Simulation Links**  
Generate a URL containing simulation parameters.

📂 **Export Results**

Download simulation data as:

- JSON
- CSV

📉 **Daily Growth Timeline**  
View portfolio value for every simulated day.

💻 **Single File Architecture**  
Runs completely in one **HTML file**.
